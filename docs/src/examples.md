# Examples

Examples are available in the [`examples/`](https://github.com/status-im/nim-chronos/tree/master/examples/) folder.

## Basic concepts

* [cancellation](https://github.com/status-im/nim-chronos/tree/master/examples/cancellation.nim) - Cancellation primer
* [timeoutsimple](https://github.com/status-im/nim-chronos/tree/master/examples/timeoutsimple.nim) - Simple timeouts
* [timeoutcomposed](https://github.com/status-im/nim-chronos/tree/master/examples/timeoutcomposed.nim) - Shared timeout of multiple tasks

## Threads

* [signalling](https://github.com/status-im/nim-chronos/tree/master/examples/signalling.nim) - Cross-thread signalling

## TCP

* [tcpserver](https://github.com/status-im/nim-chronos/tree/master/examples/tcpserver.nim) - Simple TCP/IP v4/v6 echo server
* [tcpserver2](https://github.com/status-im/nim-chronos/tree/master/examples/tcpserver.nim) - TCP/IP v4/6 echo server with graceful shutdown
* [tcpclient](https://github.com/status-im/nim-chronos/tree/master/examples/tcpserver.nim) - Simple multi-connection echo client

## HTTP

* [httpget](https://github.com/status-im/nim-chronos/tree/master/examples/httpget.nim) - Downloading a web page using the http client
* [twogets](https://github.com/status-im/nim-chronos/tree/master/examples/twogets.nim) - Download two pages concurrently
* [middleware](https://github.com/status-im/nim-chronos/tree/master/examples/middleware.nim) - Deploy multiple HTTP server middlewares
